# 1С как иностранный язык

**Для тех, кто уже умеет программировать**

Книга для разработчиков, переходящих между 1С и другими языками — Python, JavaScript, C#. База сравнения — 1С: каждая конструкция объясняется сначала как она есть в 1С и почему устроена именно так, потом — аналогии с другими языками и ловушки при переходе.

## Текущее состояние

Написано 5 глав (§ 0.1 — § 4.7). В работе: функции, обработка ошибок, асинхронность, объекты и модули.
