# Конституция книги

## Что эта книга ЕСТЬ

- Сравнительный справочник синтаксиса и архитектуры 1С для разработчиков с опытом в других языках
- База — 1С: каждая конструкция объясняется от 1С наружу, не наоборот
- Книга работает в обе стороны: и для тех, кто приходит в 1С, и для тех, кто уходит из 1С
- Исторический контекст языковых решений — несущий аргумент, не украшение

## Что эта книга НЕ ЕСТЬ

- Не учебник программирования с нуля — читатель уже умеет программировать
- Не справочник по API платформы — для этого есть синтакс-помощник и ИТС
- Не полное руководство по 1С — только синтаксис и архитектурные концепции
- Не сравнение «кто лучше» — только «чем отличается и почему»

## Принципы подачи материала

- Сначала зачем, потом как
- Каждый параграф: описание в 1С → таблица аналогий → блок ловушек
- Бытовой регистр + строгий термин в скобках при первом введении
- Метафоры с парной строгой формулировкой
- Минимум разметки, максимум прозы
- Никаких абсолютов — только «возникает когда…», «требует…»

## Технические соглашения

- Синтаксис 1С — русский (основной в реальных конфигурациях)
- Английский синтаксис не дублируется — читатель направляется к синтакс-помощнику
- Сквозной пример — торговая компания «Канцтовары» (3 сотрудника, несколько товаров, клиенты)
- Платформа — 1С:Предприятие 8.3 (последние возможности: async 8.3.18+)

## Аудитория

Первичная: разработчик на Python / JavaScript / C#, которому нужно работать с 1С.
Вторичная: разработчик на 1С, которому нужно работать с другими языками.
