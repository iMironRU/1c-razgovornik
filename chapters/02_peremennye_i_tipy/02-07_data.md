## § 2.7 Дата

Дата в 1С — встроенный примитивный тип, не библиотека. Это означает, что с датами можно работать сразу, без импортов и создания объектов.

Литерал даты записывается апострофами:

```bsl
Д1 = '20240115';           // 15 января 2024 года, время 00:00:00
Д2 = '20240115143000';     // 15 января 2024, 14:30:00
```

Арифметика с датами работает через секунды: прибавление числа к дате добавляет секунды.

```bsl
Завтра = СегодняДата + 86400;      // + одни сутки (86400 секунд)
ЧерезЧас = ТекущаяДата() + 3600;   // + один час
```

Полезные функции:

```bsl
ТекущаяДата()                   // текущая дата и время
НачалоДня(Дата)                 // дата с временем 00:00:00
КонецДня(Дата)                  // дата с временем 23:59:59
НачалоМесяца(Дата)              // первый день месяца
КонецМесяца(Дата)               // последний день месяца
ДобавитьМесяц(Дата, 3)          // + 3 месяца
РазностьДат(Д1, Д2, "День")     // количество дней между датами
Год(Дата)                       // числовой год
Месяц(Дата)                     // числовой месяц (1–12)
День(Дата)                      // числовой день
ДеньНедели(Дата)                // 1 (пн) – 7 (вс)
```

```
Операция              1С                          Python
────────────────────  ──────────────────────────  ──────────────────────────────
Создать дату          '20240115'                  datetime(2024, 1, 15)
Текущее время         ТекущаяДата()               datetime.now()
Добавить день         Дата + 86400                Дата + timedelta(days=1)
Разность в днях       РазностьДат(Д1,Д2,"День")  (Д2 - Д1).days
Начало месяца         НачалоМесяца(Дата)          Дата.replace(day=1)
```

Разработчику из Python бросится в глаза, что прибавление идёт в секундах, а не через `timedelta`. Это следствие того, что дата и время в 1С — единый тип, и секунда является его минимальной единицей. Нет отдельного типа «только дата» без времени — есть дата с временем 00:00:00.

> Дата — встроенный примитивный тип. Арифметика в секундах. Минимальная единица — секунда; типа «только дата» нет, только «дата с временем».

**Ловушки при переходе:**

*Из Python в 1С.* Инстинкт — `from datetime import datetime`. В 1С это не нужно: дата работает сразу.

*Из 1С в Python.* Инстинкт — прибавить число к дате. В Python это вызовет ошибку: нужен `timedelta`.

*Из 1С в любой другой язык.* Инстинкт — ожидать, что дата умеет всё, что умеет `ТекущаяДата()` и `РазностьДат()`. В других языках эти функции нужно искать в библиотеке или писать самостоятельно.

## Итог главы

В 1С пять примитивных типов — строка, число, булево, неопределено и дата. Единственный числовой тип покрывает и целые, и дробные значения. Строки сравниваются без учёта регистра, индексация в строковых функциях начинается с единицы. Булево не имеет неявного приведения — условие должно быть явным. Два специальных значения, `Неопределено` и `Null`, означают разные вещи: первое — отсутствие значения в коде, второе — отсутствие значения в данных из базы. Дата — встроенный примитив, работает без импортов, арифметика в секундах.

Следующая глава — про коллекции. В 1С их семь видов, и каждый занимает свою нишу.
