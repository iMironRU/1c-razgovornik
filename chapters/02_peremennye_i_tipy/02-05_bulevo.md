## § 2.5 Булево

Булев тип в 1С не содержит сюрпризов по значениям: `Истина` и `Ложь`. Логические операторы — `И`, `ИЛИ`, `НЕ` (и их английские аналоги `And`, `Or`, `Not`).

```bsl
Флаг = Истина;
Другой = Ложь;

Если Флаг И НЕ Другой Тогда
    Сообщить("условие выполнено");
КонецЕсли;
```

Главное отличие от JavaScript — в 1С **нет неявного приведения к булево** в условиях. В JavaScript любое значение имеет «истинность» (truthy/falsy): пустая строка — ложь, ноль — ложь, `null` — ложь. В 1С условие должно быть явно булевым. Написать `Если Значение Тогда` вместо `Если Значение <> "" Тогда` — ошибка.

```bsl
// 1С — так нельзя, если Строка не булево
Если Строка Тогда ... // ОШИБКА

// Нужно явно:
Если Строка <> "" Тогда ...
Если Число <> 0 Тогда ...
Если Значение <> Неопределено Тогда ...
```

```javascript
// JavaScript — так можно (truthy/falsy)
if (str) { ... }     // пустая строка — false
if (num) { ... }     // 0 — false
if (value) { ... }   // null/undefined — false
```

> В 1С нет неявного приведения к булево. Условие должно быть явным сравнением. Привычка из JavaScript писать `Если Значение Тогда` приведёт к ошибке.
