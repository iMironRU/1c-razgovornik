## § 2.4 Число

В 1С единственный числовой тип — **Число**. Нет разделения на `int` и `float`, нет `double`, нет `long`. Число в 1С — это десятичное число с высокой точностью: до 38 значащих цифр, дробная часть обычно до 10 знаков (точность зависит от контекста: настроек реквизитов, операций округления, конкретных вычислений).

```bsl
Цена = 1500;       // целое
Ставка = 0.18;     // дробное
Итого = Цена + Цена * Ставка;   // 1770
```

Для разработчика на C# отсутствие разделения на `int` и `double` поначалу кажется неточностью. На практике это удобство: в деловом учёте числа почти всегда десятичные, и отдельный тип для целых только добавлял бы преобразования.

Деление в 1С всегда дробное — нет целочисленного деления как в Python (`//`) или C# (деление двух `int`):

```bsl
10 / 3    // 3.333... (не 3)
```

Для получения целой части используется функция `Цел()`, для остатка — `?(остаток)` или функция `ОКР()`:

```bsl
Цел(10 / 3)              // 3
10 - Цел(10 / 3) * 3     // 1 (остаток через арифметику)
ОКР(3.7, 0)              // 4 (округление до целых)
ОКР(3.7, 0, 1)           // 3 (округление вниз)
```

**Ловушки при переходе:**

*Из Python в 1С.* Оператор `//` не существует. `10 // 3` вызовет ошибку синтаксиса.

*Из C# в 1С.* Деление двух чисел всегда даёт дробный результат — нет случайного целочисленного деления, которое в C# режет дробную часть при делении двух `int`.
