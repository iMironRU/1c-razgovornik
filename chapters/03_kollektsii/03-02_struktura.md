## § 3.2 Структура

Структура — коллекция пар «имя : значение», где имя всегда строка, значение — любого типа. Ближайший аналог — словарь в Python, объект в JavaScript, `Dictionary<string, object>` в C#.

Ключевое ограничение: **имя ключа должно быть допустимым идентификатором** — только буквы, цифры и подчёркивание, без пробелов. Это отличает Структуру от Соответствия, где ключ может быть любым.

```bsl
// Создание — пустая
Товар = Новый Структура;
Товар.Вставить("Наименование", "Ручка");
Товар.Вставить("Цена", 45);
Товар.Вставить("Остаток", 100);

// Создание через перечисление полей
Товар = Новый Структура("Наименование, Цена, Остаток", "Ручка", 45, 100);

// Доступ к значению
Название = Товар.Наименование;        // через точку
Цена = Товар["Цена"];                 // через скобки — тот же результат

// Проверка наличия ключа
Если Товар.Свойство("Артикул") Тогда
    // ключ существует
КонецЕсли;

// Перебор
Для Каждого Элемент Из Товар Цикл
    Сообщить(Элемент.Ключ + ": " + Элемент.Значение);
КонецЦикла;
```

| Операция | 1С Структура | Python dict | JavaScript object | C# Dictionary |
|---|---|---|---|---|
| Создать | `Новый Структура` | `{}` | `{}` | `new Dictionary<>()` |
| Добавить | `.Вставить(к, в)` | `d[k] = v` | `obj[k] = v` | `d[k] = v` |
| Получить | `.Ключ` или `["Ключ"]` | `d[k]` | `obj[k]` | `d[k]` |
| Проверить | `.Свойство("К")` | `k in d` | `k in obj` | `.ContainsKey(k)` |
| Удалить | `.Удалить("К")` | `del d[k]` | `delete obj[k]` | `.Remove(k)` |

> Ключи Структуры — только допустимые идентификаторы. Ключ с пробелом или начинающийся с цифры создать нельзя. Для произвольных ключей — Соответствие.

**Идиома 1С: Структура как контейнер параметров.**
В 1С нет функций с произвольным числом параметров (`*args`, `**kwargs`). Стандартный паттерн — передать структуру:

```bsl
// Вместо функции с множеством параметров
Функция СоздатьЗаказ(Параметры) Экспорт
    Клиент     = Параметры.Клиент;
    Сумма      = Параметры.Сумма;
    Комментарий = ?(Параметры.Свойство("Комментарий"), Параметры.Комментарий, "");
КонецФункции

// Вызов
П = Новый Структура("Клиент, Сумма", СсылкаНаКлиента, 5000);
СоздатьЗаказ(П);
```

Разработчику из Python этот паттерн знаком — он похож на передачу `**kwargs` через словарь. Разница в том, что в 1С это не синтаксический сахар, а явная передача объекта.
