## § 3.6 Дерево значений

Дерево значений — иерархическая версия таблицы значений. Каждая строка может содержать подстроки, те — свои подстроки, и так до произвольной глубины. Структура колонок задаётся так же, как в таблице значений.

```bsl
Дерево = Новый ДеревоЗначений;
Дерево.Колонки.Добавить("Наименование");
Дерево.Колонки.Добавить("Сумма");

// Корневые строки
КорневаяСтрока = Дерево.Строки.Добавить();
КорневаяСтрока.Наименование = "Канцтовары";
КорневаяСтрока.Сумма = 0;

// Дочерние строки
ДочерняяСтрока = КорневаяСтрока.Строки.Добавить();
ДочерняяСтрока.Наименование = "Ручки";
ДочерняяСтрока.Сумма = 4500;

ДочерняяСтрока = КорневаяСтрока.Строки.Добавить();
ДочерняяСтрока.Наименование = "Карандаши";
ДочерняяСтрока.Сумма = 2000;

// Перебор — рекурсивно или через встроенный обход
Для Каждого Строка Из Дерево.Строки Цикл
    Сообщить(Строка.Наименование);
    Для Каждого ДочСтрока Из Строка.Строки Цикл
        Сообщить("  " + ДочСтрока.Наименование);
    КонецЦикла;
КонецЦикла;
```

Дерево значений применяется там, где данные сами по себе иерархические: категории товаров, организационная структура, бюджетные статьи с подстатьями. Прямого аналога в стандартных библиотеках других языков нет — там аналогичные структуры строятся вручную через рекурсивные классы или словари словарей.

```python
# Python — ближайший аналог: словарь словарей
дерево = {
    "Канцтовары": {
        "сумма": 0,
        "дети": {
            "Ручки":     {"сумма": 4500, "дети": {}},
            "Карандаши": {"сумма": 2000, "дети": {}},
        }
    }
}
```

> Дерево значений — иерархическая таблица: строки содержат подстроки произвольной глубины. Применяется для представления категорий, структур, вложенных группировок. Как и таблица значений, является серверным типом.
