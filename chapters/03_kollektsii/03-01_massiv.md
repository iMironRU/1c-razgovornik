## § 3.1 Массив

Массив в 1С — индексированная коллекция значений произвольного типа. Индексация с нуля. Элементы могут быть разного типа — никакой типизации массива нет.

```bsl
// Создание
Товары = Новый Массив;
Товары.Добавить("Ручка");
Товары.Добавить("Карандаш");
Товары.Добавить(100);          // разные типы — допустимо

// Создание с начальным размером
Буфер = Новый Массив(10);      // 10 элементов, все Неопределено

// Доступ по индексу
Первый = Товары[0];            // "Ручка"
Товары[1] = "Фломастер";       // замена

// Основные методы
Товары.Добавить("Линейка");                    // в конец
Товары.Вставить(1, "Скрепка");                // по индексу
Товары.Удалить(0);                             // по индексу
КолВо = Товары.Количество();                   // размер
Товары.Очистить();                             // очистить
Индекс = Товары.Найти("Карандаш");            // -1 если не найден
```

Перебор — через цикл по коллекции или через счётчик:

```bsl
// По коллекции — предпочтительно
Для Каждого Товар Из Товары Цикл
    Сообщить(Товар);
КонецЦикла;

// По индексу — когда нужен номер
Для Сч = 0 По Товары.Количество() - 1 Цикл
    Сообщить(Товары[Сч]);
КонецЦикла;
```

| Операция | 1С | Python | JavaScript | C# |
|---|---|---|---|---|
| Создать | `Новый Массив` | `[]` | `[]` | `new List<T>()` |
| Добавить | `.Добавить(х)` | `.append(x)` | `.push(x)` | `.Add(x)` |
| Вставить | `.Вставить(i, х)` | `.insert(i, x)` | `.splice(i,0,x)` | `.Insert(i, x)` |
| Удалить | `.Удалить(i)` | `.pop(i)` | `.splice(i,1)` | `.RemoveAt(i)` |
| Размер | `.Количество()` | `len(a)` | `.length` | `.Count` |
| Найти | `.Найти(х)` | `.index(x)` | `.indexOf(x)` | `.IndexOf(x)` |

> Индексация с нуля. Метод `.Найти()` возвращает `Неопределено` если элемент не найден — не `-1`, как в большинстве других языков.

**Ловушки при переходе:**

*Из Python в 1С.* `.Найти()` возвращает `Неопределено`, а не `-1`. Проверка `Если Индекс = -1` не сработает — нужно `Если Индекс = Неопределено`.

*Из JavaScript в 1С.* Нет методов `.map()`, `.filter()`, `.reduce()`. Всё через явные циклы.

*Из 1С в Python.* `len()` — функция, а не метод. `товары.Количество()` → `len(товары)`.
